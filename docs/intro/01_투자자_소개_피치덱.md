## 2. 문제 — 트레이더는 분석이 아니라 결정에서 무너진다

차트는 잘 본다. 그런데 진입 버튼 앞에서 흔들린다.

| 실패 패턴 | 평균 손실 |
|---|---|
| **추격 진입** — 이미 움직인 뒤 따라 들어감, 손익비 붕괴 | −1.2R / 거래 |
| **손절 미준수** — 계획한 손절가에서 한 번 버팀 | −2.5R / 사고 |
| **사이즈 오버** — 확신이 강할수록 베팅이 커짐 | 최대 −50% / 계좌 |

> 트레이더가 잃는 진짜 이유는 분석 부족이 아니라 **결정의 일관성 붕괴**다.
> 시장에는 "차트 보는 도구"는 넘치지만, "지금 이 거래를 해도 되는가"를 답하는 도구는 없다.

---

## 3. 솔루션 — 진입 전 5분, 결정을 시스템화

매번 같은 흐름으로 거래하면 감정이 끼어들 자리가 없다.

**4단계 의사결정 루프**

1. **AI 분석** (진입 전) — Binance 실시간 데이터로 시장 구조·수급·심리 분석. 시나리오 1~3개 + 무효화 조건 제시.
2. **주문 검토** (진입 전) — 진입가·손절·목표를 입력하면 **A·B·C·D 등급**으로 답. 추격·미확정 캔들·BTC 충돌 자동 감지.
3. **내 거래** (진입 후) — 진입 평가가 영구 보존. 청산 결과 입력 시 AI가 한국어 복기 코멘트 자동 생성.
4. **성과 분석** (진입 후) — 등급별 평균 R·승률·실수 태그 통계로 패턴을 객관화.

> 핵심 가치: **한 번의 D급 거래가 한 달 수익을 지운다. 그 거래를 진입 전에 막는다.**

---

## 4. 제품 데모 — 어떻게 작동하나

- **AI 분석 결과**: 시나리오 카드 (A 롱 / B 숏 / C 관망) + 트리거·무효화·실효 R:R
- **주문 검토**: 등급 배지(D = 강한 자제) + 점수 분해(손절 구조 +2 / BTC 정렬 +1 / 계획 구간 벗어남 −2)
- **AI 복기**: 청산 후 한국어 코칭 코멘트 자동 생성
- **성과 대시보드**: 등급별 평균 R 비교 (A +1.4R / B +0.6R / C −0.4R / D −1.8R)

> (각 화면 캡처 4장 배치)

## 5. 차별점 — 왜 우리가 이긴다

| | 일반 차트 도구 (TradingView 등) | 저널 앱 | **Alpha Gate** |
|---|---|---|---|
| 차트 표시 | ✅ | — | ✅ |
| **진입 여부 판정(등급)** | ❌ | ❌ | ✅ |
| 객관 데이터 기반 AI | ❌ | ❌ | ✅ |
| 자동 복기 코칭 | ❌ | 수동 기록만 | ✅ |
| 진입 평가 ↔ 결과 연결 | ❌ | ❌ | ✅ |

세 가지 본질적 차별점:
1. **분석이 아니라 결정** — 차트에서 "진입할지 말지"를 점수로 답한다.
2. **객관 데이터 기반** — LLM에 가격을 직접 묻지 않는다. Binance 실데이터를 수집한 뒤 *해석만* AI에 맡긴다 (환각 차단).
3. **결정과 결과를 잇는다** — 진입 등급 → 청산 결과 → AI 복기가 한 흐름으로 연결돼 학습 루프를 만든다.

## 6. 기술 해자 — 3단계 AI 파이프라인

단일 LLM 호출이 아니라 책임을 분리한 에이전트 구조.

1. **Stage 1 — Market Data (결정론적 코드)**: Binance 멀티 TF 캔들·호가·체결·펀딩·OI·도미넌스 병렬 수집 + 자체 계산(스윙·FVG·오더블록·유동성·볼륨 프로파일·오더플로우).
2. **Stage 2 — Strategy Agent (경량 LLM)**: 5개 전략 중 1개 선택 (추세눌림/돌파/박스/반전/관망).
3. **Stage 3 — Scenario Generator (메인 LLM)**: 선택된 전략 안에서만 시나리오 생성 + 스타일별 손절/목표/RR 표준 강제.

> 데이터가 ground truth, AI는 해석자. → **검증 가능하고 환각에 강한 분석**.
> 프롬프트 캐싱으로 호출 비용 최적화. 스타일(스캘핑~포지션)별 표준 손절/목표 룰 내장.

## 7. 시장 (TAM / SAM / SOM)

- **거시 추세**: 전 세계 암호화폐 보유자 증가, 24/7 무기한 선물 시장의 리테일 거래자 폭증.
- **TAM**: 글로벌 암호화폐 액티브 트레이더 (수천만 명 규모).
- **SAM**: 한국어권 + 진지하게 손익 관리하는 액티브 선물 트레이더.
- **SOM (초기)**: Victor Alpha 블로그 독자층 + 트레이딩 교육/저널 수요층.

> ⚠️ 구체 수치는 출처(Statista / 거래소 공시 / 설문)로 백업 필요. 현재는 정성 추정.

| 플랜 | 가격 | 월 AI 분석 |
|---|---|---|
| Free | ₩0 (영구 무료) | 5회 |
| Standard | ₩15,000 / 월 | 100회 |
| Pro | ₩95,000 / 월 | 500회 |
| Premium | ₩295,000 / 월 | 무제한 |

- **추가 수익원**: AI 크레딧 단건 결제(vUSDT), (로드맵) Victor Alpha 블로그 유료 구독 번들 + Toss 결제.
- **마진 구조**: 변동비는 Claude API 호출이 대부분 → 프롬프트 캐싱 + 횟수 제한으로 통제.
- **확장**: (로드맵) 실거래 API 연동 시 거래 수수료 레브쉐어/프리미엄 가능성.
