# Alpha Gate — 사용자 가이드

> **"무엇을 살까"가 아니라 "이 거래를 해도 되는가"에 답하는 도구.**
> 진입 버튼을 누르기 전에, 이 거래가 통계적으로 할 만한 거래인지 5분 만에 검증합니다.

이 문서는 Alpha Gate를 **실제로 사용하는 방법**을 단계별로 설명합니다. 시스템 내부 동작·아키텍처는 [시스템 상세](시스템-상세.md) 문서를 참고하세요.

---

## 1. Alpha Gate는 무엇인가

대부분의 손실은 **차트 실력**이 아니라 **진입 순간의 결정**에서 생깁니다.

- 차트는 봤는데 막상 진입하니 추격 매수가 되어 있다
- 손절가를 정해놨는데 "이번 한 번만" 버티다 한 달 수익을 날린다
- 확신이 강해서 평소보다 크게 베팅했다가 크게 잃는다

Alpha Gate는 그 **진입 순간을 객관화**합니다. AI가 가격을 지어내지 않고, Binance 실시간 데이터를 해석해 시나리오를 제시하고, 그 거래를 등급으로 채점해 "지금 해도 되는지"를 알려줍니다.

> ⚠️ **본 서비스는 투자 자문이 아닙니다.** 모든 매매 결정과 결과는 사용자 본인의 책임입니다.

---

## 2. 전체 흐름 (한눈에)

```
시장 현황 ──► AI 분석 ──► 거래 실행 ──► 가상 거래 ──► 거래 일지 ──► 성과 분석
 (지금 장이      (코인·스타일     (등급 확인 후    (실제 자금       (결과 기록 +    (복기 통계)
  볼 만한가)      선택 → 시나리오)   진입 결정)      없이 체결 학습)   AI 복기)
```

좌측 사이드바의 메뉴 순서가 곧 권장 워크플로우입니다. 처음이면 **AI 분석 → 거래 실행 → 가상 거래** 순서대로 한 바퀴 돌려보세요.

---

## 3. 화면별 사용법

### 3.1 후보 레이더 — "지금 볼 만한 코인"

AI 분석 페이지 상단에 있습니다. **"무엇을 살까"가 아니라 "분석해볼 후보"**를 코드 신호로 골라줍니다 (LLM 비용 없이 결정론적).

- **트레이딩 스타일 탭**(스캘핑/데이/스윙/포지션)을 고르면, 그 스타일로 **진입 가능한 코인**만 점수순으로 보여줍니다.
- 각 행 정보:
  - **추세 마크** ▲상승 / ▼하락 / ━박스권
  - **예상 방향** — 롱 예상 / 숏 예상 / 양방향. *추세 기반 추정*이며, 실제 방향은 분석을 돌려야 확정됩니다. (양방향 = 방향 불명확)
  - **신호 칩** — 유동성 sweep, 변동성 수축, 매물대 도달, 펀딩 극단 등
  - **예상 손절폭 · 목표폭(%)**, **예상 변동폭(±%)**, **점수**, **예상 시나리오 개수**
- **BTC·ETH·XRP·BNB는 항상 맨 앞에 고정**되어 있어 언제든 분석할 수 있습니다.
- 행을 클릭하면 그 코인 + 추천 스타일이 분석 폼에 자동 입력됩니다. (분석은 **수동 실행** — AI 크레딧 보호)

> 레이더는 10분마다 자동 스캔되며, 가격·변동률은 25초마다 갱신됩니다. 직접 **새로고침**을 눌러 즉시 재스캔할 수도 있습니다(5분 레이트리밋).

### 3.2 AI 분석 — 진입 전에 시장을 읽는다

1. **코인**과 **트레이딩 스타일**을 고릅니다.
2. (선택) **운영 자금 / 거래당 리스크 %**를 조정합니다. 비우면 프로필 기본값을 씁니다.
3. **[분석 실행]** → 약 10~20초 후 결과가 나옵니다.

**결과 읽는 법:**
- **요약 + 시장 구조** — 추세, 핵심 레벨, 수급 흐름
- **시나리오 1~3개** — 각각 **진입 조건(trigger) · 진입가 · 손절 · 목표 · 예상 손익비**
  - 시나리오마다 **과거 적중률**과 **도달 확률(목표/손절/미도달)** 표시
- **AI 자신감(confidence)** + **전략 종류**(추세 눌림목 / 돌파 / 박스권 / 반전 / 관망)
- 진입할 시나리오의 **[실행]** 버튼 → 거래 실행 페이지로 모든 값이 넘어갑니다.

> 분석 결과는 자동으로 영구 저장됩니다. 나중에 대시보드·기록에서 다시 열어볼 수 있습니다.

### 3.3 거래 실행 — "이 거래를 해도 되는가"

진입 전 마지막 관문. 셋업·자금·시장 컨텍스트를 한 번에 평가합니다.

- **① 요약바** — 등급 결론(예: "조건부 진입 — 6점") + 심볼·방향·레버리지·진입·목표·손절·**실효 손익비**(수수료 차감 후 R) + ✓통과 / ⚠경고 개수
- **② 주문 정보**
  - 심볼 · 롱/숏 · **주문 방식 2택** · 레버리지
  - **지금 바로**(시장가, 즉시 체결) / **예약 주문 — 자동**
    - 예약 주문은 **지정가↔역지정가를 앱이 자동 판정**합니다. 예약가가 현재가보다 유리한 쪽이면 "되돌림 대기(지정가)", 불리한 쪽이면 "돌파 추격(역지정가)". 예약가를 옮기면 실시간으로 바뀝니다. (전문 용어를 몰라도 됩니다)
  - 진입가 / 손절가(−%) / 목표가(+%) + 사이즈 바 + 주문 금액 + 청산가(대략)
- **③ 자동 점검**(접힘) — 자금 관리 상태, 시장 컨텍스트(BTC·펀딩), 시장 구조 체크. AI와 일지에서 자동으로 채워집니다. 문제가 있으면 ⚠ 경고 배지가 뜹니다.
- **④ 어떻게 실행할까요?**(실행 3택)
  - **가상 거래로 실행(vUSDT)** — 실제 자금 없이 동일한 체결·수수료·마진으로 학습 (현재 메인 경로)
  - **실거래 주문** — *준비 중* (거래소 자동 주문 인프라 도입 후 활성화)
  - **알림만 등록하고 대기** — 지금 진입하지 않고, 가격이 시나리오 레벨(진입/손절/목표)에 도달하면 텔레그램으로 알림

**등급별 권장 행동:**

| 등급 | 의미 | 권장 |
|------|------|------|
| **A** (≥8점) | 좋은 거래 | 계획대로 진입. 손절가는 반드시 지키기 |
| **B** (5~7점) | 조건부 진입 | 진입 가능하나 포지션을 평소의 절반으로 |
| **C** (2~4점) | 비추천 | 눌림 대기 권장. 굳이 한다면 축소 |
| **D** (≤1점) | 강한 자제 | 하드 차단은 안 하지만, 컨펌을 한 번 더 받습니다. 셋업 문제면 10% 축소, 계좌 문제(누적 손실/과노출)면 "오늘은 보류" |

> D등급도 막지는 않습니다 — 경고 + 확인 후 진행할 수 있습니다. 단, **일일 손실 한도(-2R)**에 도달하면 강하게 자제를 권합니다.

### 3.4 가상 거래 — 거래소처럼 연습

실제 자금 없이 **거래소와 동일한 흐름**(체결가·슬리피지·수수료·마진)으로 포지션을 운용합니다. vUSDT(가상 USDT) 지갑을 씁니다.

- 거래소 스타일 주문 패널 (시장가/지정가/역지정가)
- 진행 중 포지션 · 대기 주문 추적, 미실현 손익 표시
- 예약(지정/역지정) 주문은 가격 도달 시 **자동 체결**, 만료(기본 24시간) 시 **자동 취소**
- 진입한 포지션은 손절/목표 도달 또는 만료 시 **자동 정산**

### 3.5 거래 일지 — 기록과 AI 복기

- 탭: **실거래 / 가상 거래 / 백테스트** 3분류
- 종료된 거래의 결과 R, 청산 사유(목표/손절/시간), 진입/청산가
- **실수 태그**(추격, 손절 미준수 등) + 메모를 남기고, **AI 코칭**으로 복기를 받을 수 있습니다

### 3.6 성과 분석 — 복기 통계

승률, 누적 R, 등급별 분포, 전략별 적중률 등을 실거래/가상/백테스트별로 봅니다.

### 3.7 알림 설정

- **텔레그램** — 봇 딥링크로 한 번에 연결 (START 클릭 시 자동 등록)
- **디스코드** — 웹훅 URL 등록
- 이벤트별 토글: D등급 경고, 연속 손실, AI 복기 완료, 일일 요약, **분석 추천 시각**, **시나리오 가격 도달**

## 4. 트레이딩 스타일 (4종)

| 스타일 | 보유 기간 | 분석 TF (장기/중기/단기) | 손절폭 | 목표폭 | 최소 손익비 |
|--------|----------|------------------------|--------|--------|------------|
| **스캘핑** | 수분~수시간 | 1H / 15M / 5M | 0.3~1.2% | 0.4~2% | 1.3+ |
| **데이** | 수시간~하루 | 4H / 1H / 15M | 0.7~1.5% | 1.5~3% | 1.5+ |
| **스윙** ⭐기본 | 며칠~수주 | 1D / 4H / 1H | 2~5% | 5~15% | 2+ |
| **포지션** | 수주~수개월 | 1D / 4H / 1H | 5~15% | 15~50% | 3+ |

> 손절/목표 범위는 **수수료(왕복 0.075% 가정) 차감 후 현실판**입니다. 스타일을 벗어나는 손절·손익비는 AI가 거부하거나 "관망"으로 안내합니다.

## 5. 자주 묻는 질문

**Q. AI가 가격을 예측하나요?**
아니요. 실제 Binance 데이터를 해석해 시나리오와 조건을 제시할 뿐, 가격을 지어내지 않습니다. "예상 변동폭"도 방향 예측이 아니라 변동성 기반 범위입니다.

**Q. 레이더의 "예상 방향"은 믿을 만한가요?**
추세 기반 추정입니다. 실거래 분석 데이터로 검증한 결과 **추세가 방향성을 가질 때 실제 분석 방향과 약 98% 일치**합니다. 다만 박스권(양방향)일 때는 방향을 단정하지 않습니다. 최종 방향은 분석을 돌려야 확정됩니다.

**Q. 가상 거래와 실거래의 차이는?**
가상 거래는 vUSDT(가짜 자금)로 동일한 체결 흐름을 학습합니다. 실거래(거래소 자동 주문)는 현재 준비 중입니다.

**Q. 예약 주문에서 지정가/역지정가를 직접 골라야 하나요?**
아니요. "예약 주문 — 자동"이 예약가와 현재가의 방향을 보고 자동으로 결정합니다.

**Q. D등급이면 거래를 못 하나요?**
하드 차단하지 않습니다. 경고와 확인 후 진행할 수 있습니다. 단, 일일 손실 한도 도달 시에는 강하게 자제를 권합니다.

## 6. 권장 루틴

1. **시장 현황**으로 BTC·세션·전반 분위기 확인
2. **후보 레이더**에서 스타일을 골라 진입 가능 코인 스캔
3. 관심 코인 **AI 분석** 실행 → 시나리오·등급 확인
4. **거래 실행**에서 등급·자금·시장 컨텍스트 점검 → 가상 진입 또는 알림 등록
5. 종료 후 **거래 일지**에 기록 + AI 복기
6. 주기적으로 **성과 분석**으로 패턴 점검
